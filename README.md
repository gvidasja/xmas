# xmas
XSaaS - Christmas Spirit as a Service

## Local development

Prerequisites:
* Deno
* VS Code recommended with extensions - Rest Client, Deno

To launch the project:

via VS Code:
1. Press `F5`

via command line:
1. `deno run --allow-all --unstable src/server/server.ts`

via docker:
1. `docker-compose up`
